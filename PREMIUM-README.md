# UroMed Clinic — Premium Medical Reference

[![Deploy to Cloudflare Pages](https://img.shields.io/badge/Cloudflare-Pages-orange?logo=cloudflare)](https://uromed-clinic.pages.dev)
[![Deploy to Vercel](https://img.shields.io/badge/Vercel-Deploy-black?logo=vercel)](https://uromed-clinic.vercel.app)

Medical reference platform for urology and andrology with EAU, AUA, RKR, UA clinical guidelines.

## ✨ Premium Features 2026

- **197 disease entries** with complete clinical content
- **World-class design system** inspired by Linear, Stripe, Apple
- **Glassmorphism surfaces** with colored lighting shadows
- **iPhone 15-17 Pro optimization** — Dynamic Island, Safe Areas
- **WCAG AA accessibility** with high contrast mode
- **Z-index consolidated** — Conflict-free modal stacking
- **PWA support** — Works offline, installable
